## ⚙️ Installation
1. Install **XAMPP** and start Apache & MySQL.
2. Copy the `vms/` folder into `htdocs/`.
3. Import the database:
   - Open phpMyAdmin.
   - Create a database named `veterinary_management`.
   - Import `vms.sql` (included in this package).
4. Open the system in your browser:  
   `http://localhost/vms`
